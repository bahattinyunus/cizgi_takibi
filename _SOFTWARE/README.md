# Software Hub: The Interface

Code implementations for various platforms and simulation environments.

## Embedded (C++/Arduino)
- `main.ino`: Core control loop.
- `PID.cpp`: Modular PID library implementation.
- `Sensors.h`: Driver logic for QTR arrays.

## Computer Vision (Python/OpenCV)
- `tracker.py`: Real-time path extraction from camera frames.
- `simulation.py`: Physics-based simulator for algorithm testing.

## Getting Started
1. Install [Arduino IDE](https://www.arduino.cc/en/software).
2. Install Python 3.10+ and `pip install opencv-python numpy`.
3. Upload the firmware to your microcontroller (ESP32 / Arduino Nano).
